# Estudo de Python ![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

Este repositório centraliza meus estudos práticos de Python, com códigos simples, porém eficazes que realizei aplicando conceitos da linguagem de programação. 
O conteúdo é desenvolvido com base nas formações da Alura, aplicando conceitos fundamentais, como:

- Estruturas de Dados: Listas, Tuplas, Conjuntos e Dicionários.

- Lógica de Programação: Condicionais e Laços de repetição.

- Performance: Introdução à Programação Assíncrona.

- Organização: Códigos modulares e testados para garantir clareza e funcionalidade.
